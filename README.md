# MagicCube v1.0.0

> 计算机图形学小组作业

**组别：** 第九组

**小组成员：** 姚昭圻，黄健军， 陈舒仪

**配置环境：** Visual Studio 2015 + freeglut

**所使用的技术：**

* 摄像机漫游(球面)
* 纹理映射
* 简单光照
* 深度测试

**程序相关操作：**

使用VS2015编译并运行项目，运行可执行程序：

键盘操作：

* 左SHIFT：切换魔方的面（前，左，上）
* 键盘左右键：在当前面切换层
* 键盘上下键：顺时针/逆时针旋转选中层
* 左CTRL：激活/关闭摄像机漫游
* w，s，a，d：摄像机漫游
* t：打开/关闭纹理映射

鼠标操作：

* 鼠标右键选择魔方阶数
